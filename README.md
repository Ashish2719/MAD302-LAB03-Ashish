## Description
This Android app requests Camera permission and simulates fetching data using a 2-second delay. The result is shown on the screen, and errors are handled properly.

## Features
- Fetch Data button
- Camera permission request
- Coroutine with delay(2000)
- Success and error messages
- Loading indicator

## How it works
User clicks the button, permission is checked. If granted, the app waits 2 seconds and then shows success or error message.

## Tools Used
Kotlin, Android Studio, ViewBinding, Coroutines
